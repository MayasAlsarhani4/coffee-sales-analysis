# Coffee Sales Analysis

In this project I used pandas to work with a coffee shop sales dataset.

I explored the dataset by checking:
- columns
- first and last rows
- shape of the dataset
- dataset information
- statistical summary

I also worked with different pandas functions like:
- selecting columns
- filtering rows
- sorting sales values
- checking missing values
- finding unique values
- removing duplicates
- renaming columns

At the end, I created a new CSV file using the first 50 rows from the dataset.

Files used:
- analysis.py
- orders.csv
